# 🌱 AgroVision
## 📖 Sobre o Projeto

Este projeto foi desenvolvido para o **Programa Agrinho 2026**, com o tema **"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"**.

O objetivo é conscientizar os visitantes sobre a importância do agronegócio sustentável, demonstrando como a tecnologia, a inovação e as práticas responsáveis podem contribuir para a preservação dos recursos naturais, garantindo qualidade de vida para as gerações atuais e futuras.

A página apresenta informações educativas, recursos interativos e ferramentas de acessibilidade, proporcionando uma experiência inclusiva e agradável para todos os usuários.

---

## 🎯 Objetivo do Tema Agrinho

O tema **"Agro forte, futuro sustentável"** busca destacar que é possível aumentar a produtividade agrícola sem comprometer o meio ambiente. O projeto incentiva reflexões sobre:

* Uso consciente dos recursos naturais;
* Preservação da biodiversidade;
* Adoção de tecnologias sustentáveis no campo;
* Importância do produtor rural para a sociedade;
* Responsabilidade ambiental e social na produção de alimentos.

---

## 🚀 Como utilizar

1. Acesse o projeto através do navegador:

   **https://vivi1518.github.io/agrinho-2.0/**

2. Navegue pelas seções da página utilizando o menu principal.

3. Explore os conteúdos educativos sobre sustentabilidade e inovação no campo.

4. Utilize os recursos de acessibilidade disponíveis na lateral direita da tela:

   * ➕ Aumentar fonte;
   * ➖ Diminuir fonte;
   * 🌙☀️ Alternar entre modo escuro e claro;
   * 🔊 Iniciar leitura por voz do conteúdo principal;
   * ⏹️ Interromper a leitura.

---

## ♿ Recursos de Acessibilidade

Com o objetivo de promover inclusão digital, o site conta com funcionalidades que facilitam o acesso às informações por diferentes públicos:

* Controle do tamanho da fonte;
* Alternância entre tema claro e escuro;
* Leitura automática do conteúdo principal;
* Botão para interromper a leitura a qualquer momento;
* Interface responsiva para diferentes tamanhos de tela.

---

## 💻 Tecnologias Utilizadas

O desenvolvimento do projeto utilizou as seguintes tecnologias:

* **HTML5** – estruturação do conteúdo;
* **CSS3** – estilização e responsividade;
* **JavaScript** – interatividade e funcionalidades dinâmicas;
* **SpeechSynthesis API** – recurso nativo do navegador para leitura por voz;
* **Git e GitHub** – versionamento e hospedagem do código;
* **GitHub Pages** – publicação do site na web.
